一、论文组成结构

妥妥八股文章，这个是按照顺序写的，分为：


  • 前置部分

  • 正文

  • 参考文献

前置部分又包括：


  • 封面

  • 诚信书

  • 致谢

  • 摘要和关键词（中文）

  • 摘要和关键词（英文）

  • 目录（中文）

  • 目录（英文）

正文的第一章应为引言，然后就是你的论文内容了，章节按一二三排下去就好了。

二、页眉页脚与目录

单独拿出来是因为它让我很头痛，翻看了诸多“百度经验”后，才逐渐弄明白页眉页脚是怎么回事。什么奇偶页不同，什么链接到上一节，什么页码从封面就开始排了，什么目录怎么自动生成。

1、目录自动生成

首先要知道，机器没有那么智能，你没有告诉它哪段文字是章节名，它就不会知道，它也很无奈啊，word有深度语义分析吗？

切换回页面视角突然发现每个章节名前面居然有黑点：

不用方，点右上角的“快速样式”里面的就好，这样不会把级别覆盖掉的。

1.2 生成目录

“引用”栏点“目录”加入自动目录就好了

，因为在1.1里已经区分了章节级别，所以自动目录会精准地显示层次出来。

目录标题左侧又有小黑点，参见1.1里的快速样式去黑点。

1.3 英文目录

在粘一份中文目录，然后把内容改成英文的就可以了。

2、页眉页脚

很难说明，我好想犯懒。

这里需要知道“节”的概念，不想知道（比如我）那就直接跳过概念进入方法步骤。

2.1 添加分节符

将光标放到你想要分节的那一页的页末，点击“页面布局”栏的“分隔符”的“分节符”的“下一页”：，随之而来的是第二页页头多了一行，用delete键删掉就好了，这样就将全文从一节变成了两节，即目录为第一节，第二页的内容为第二节。

2.2 将每章都分节

如题（想写“如本节标题”，但又会和上文的“节”发生歧义，仔细想想，“节”的意思似乎就是这个），就这样，不好截图，我就放了一个全局图示。改好了后，在大纲视图下，应该会出现下图和谐的场景，注意，虚线代表的就是分节符。

2.3 添加页眉

因为要求是“奇数页为章节名，偶数页为论文题目”，所以首先奇偶页必须分开，如果不分节，强行添加页眉的话，就会出现下列尴尬情况：

第一章的页眉我写好了，开始写第二章的页眉，写好第二章的页眉，转眼一看，第一章的页眉的内容变成了第二章的页眉的内容。

双击页面上部，或者点“插入”栏的“页眉”，页眉处左侧会出现“页眉-第x节-”字样、左侧会出现“与上节相同”的字样（或者没有），如果不同章节的页眉显示不同的节数，那么恭喜你，离成功就只剩几步了。

在页眉编辑打开的情况下，选取“奇偶页不同”，“链接到前一条页眉”的意思就是说，与上节页眉的格式、内容相同，如果你不嫌麻烦，就全部关闭“链接到前一条页眉”，这样会使你的所有页眉都能单独编辑，不会出现不同页的页眉同时变化的尴尬事。当然，图方便，就把每一章的后续页眉均打开“链接到前一条页眉”，注意是每一章的后续页，每一章的第一页必须关闭“链接到前一条页眉”选项的。

2.4 添加页脚

由于页码是加在正文与参考文献里面的（见一、论文组成结构），前面的内容不该有页码。双击页面下部，打开页脚编辑，点击 “目录”页后的页面的页脚（不出意外应该是引言部分）以及下一节的页脚，关闭“链接到前一条页眉”，（值得注意的是，此处的“链接到前一条页眉”的“页眉”指的就是页脚，不要弄迷糊了），然后将后续页的“链接到前一条页眉”均打开，再次回到“目录”页的后续页面（引言，可能），点击“设计”栏的“页码”，设置“页码格式”，改“起始页码”为1，引言部分的下一节（不出意外应该是“第一章”），再次加入页码，就可以了。

这里的步骤很难截图，就仔细阅读步骤吧。

2.5 更新目录页码

点“更新目录”的“只更新页码”就ok了。

三、公式编号

用三栏表格就能搞定，左边、右边占15%，中间占70%，中间写公式，右边写右对齐编号，最后把表格的边框给去掉就好。
